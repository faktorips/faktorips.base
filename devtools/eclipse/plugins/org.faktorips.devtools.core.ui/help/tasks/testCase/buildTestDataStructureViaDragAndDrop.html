Test Datenstruktur per Drag and Drop aufbauen

Eine Test Datenstruktur kann mittels Drag and Drop aufgebaut werden. Produktbausteine können von beliebigen Explorern aus in den Testfalleditor gezogen werden.

Hinweis

Ein Drag and Drop aus dem Produktstruktur-Explorer heraus ist zum momentanen Zeitpunkt noch nicht möglich.

Öffnen Sie die Produktdefinitions-Perspektive und legen Sie einen neuen Testfall an.

Ziehen Sie einen Produktbaustein vom Produktdefinitions-Explorer auf den Wurzel-Knoten im Testfall-Editor. Es muss sich um einen Produktbaustein handeln, der für die entsprechende Beziehung qualifiziert ist. Es ist nicht möglich einen ungültigen Produktbaustein fallen zu lassen.

Hinweis

Es kann immer nur ein Produktbaustein gleichzeitig mittels Drag and Drop eingehangen werden.
